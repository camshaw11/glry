# GLRY

  Glry is a website that allows you to search for an image based on a query.
  
  Users can click on one of the suggested queries to get started.
  
  
  # View the live site 
  
  http://gllry.herokuapp.com/#/
  
  
 ## To get started:
 
  1) Clone the repo
  
  2) Run npm install
